# Fieldstone Context

## Domain Terms

- **Collection**: A named content group declared by an app in `collections/*.ts`.
- **Field**: A named value definition inside a collection. Text fields are the current field capability.
- **Document**: A stored record in a collection, including user fields plus system fields.
- **Schema plan**: The compiled Fieldstone description of collections, fields, system fields, generated identifiers, and validation results. Runtime schema, generated schema source, generated types, and fingerprints render from this plan.
